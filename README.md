## ✨ 核心特性

### 🔒 绝对的隐私与安全

Jent 坚信日记是用户最私密的财产：

零收集、零传输：本项目不会保存、访问或收集任何用户隐私。数据交互和内容渲染完全在您的本地浏览器中完成。（注：基础的网页访问量统计由 Cloudflare 默认分析服务提供，开发者不获取任何业务级数据。）
数据由你掌控 (BYOS)：目前需授权您个人的 GitHub 账号，将您的日记文件直接同步至您的私有仓库中。后续也会提供 S3 / WebDAV 等更多云存储协议的支持。
端到端加密护航：针对同步到远程仓库的数据，我们支持生成强加密的压缩包模式。这一加密在本地的 Service Worker 中完成（依赖现代浏览器支持的 HTTPS 环境），确保即使云端被攻破，您的日记依然绝对安全。
(提示：为了保证应用的高性能离线读取，数据在您本机的浏览器 IndexedDB 中暂为明文存储，请保管好私人设备。)

### 📱 极致的全端及 PWA 体验

为了让 Web 获得媲美原生的体验，我们做了深度优化：

多端自适应布局：完美适配不同设备的屏幕尺寸——移动端（单栏视图）、平板端（双栏视图）、桌面电脑端（三栏全景视图）。
原生级触摸手势：针对手机和平板专门加入了大量触摸手势交互（如流畅的滑动切换、跟手动画等）。
离线使用能力：作为一款标准的 PWA 应用，在首次加载并缓存必要资源后，即使在完全没有网络的环境下，您依然可以流畅地查阅和撰写日记。
